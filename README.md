# hyScratch

Monorepo inicial para MVP de avaliacao automatica com Scratch.

## Estrutura

- apps/web: frontend da aula e area do professor
- apps/api: API para tentativas, score e progresso
- packages/evaluator: motor de avaliacao estrutural/comportamental

## Como rodar

1. npm install
2. npm run dev:web
3. npm run dev:api
